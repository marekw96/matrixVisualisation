clearCubes - remove all matrixes
create <name> <x> <y> <z> as <color> - makes matrix
setAllCubesStyle <name> <style> - changes all cubes in matrix to specified style:normal, hidden, selected
wait <time_s> - blocks script to some time
setPosition <name> <x> <y> <z> - sets matrix position
rotate <name> <x> <y> <z> - rotates matrix in degrees
changeBlockStyle <matrix_name> <x> <y> <z> to <style> - change given cube to style: normal, hidden, selected
